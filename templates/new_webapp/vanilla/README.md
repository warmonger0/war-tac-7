# My App - Vanilla JavaScript Template

This project was created from the **tac-webbuilder Vanilla JavaScript template** and is pre-configured for ADW-powered development.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS variables
- **JavaScript (ES6+)** - Modern JavaScript features
- **No build step** - Runs directly in the browser

## Getting Started

### Prerequisites

- A modern web browser
- A local development server (optional)

### Running the App

#### Option 1: Open Directly

Simply open `index.html` in your web browser.

#### Option 2: Local Development Server

Using Python:
```bash
python -m http.server 8000
```

Using Node.js:
```bash
npx http-server
```

Using PHP:
```bash
php -S localhost:8000
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.

## Development

### Adding New Features

1. Edit `index.html` to add HTML elements
2. Add styles to `style.css`
3. Add JavaScript logic to `script.js`
4. Reload the page to see changes

### Best Practices

- Use semantic HTML5 elements
- Keep CSS organized with CSS variables
- Use modern JavaScript features (const, let, arrow functions)
- Add comments to explain complex logic
- Keep files small and focused
